# Для запуска прграммы необходимо запустить main.py
```
python3 main.py
```
Затем необходимо перейти на localhost

## Data collector

На стартовой странице в появившейся форму можно ввести тикеры одной или двух компаний и нажать find information

Примеры названий
* AAPL  (Apple)
* MSFT (Microsoft)
* FB (Facebook)
* AMZN (Amazon)
